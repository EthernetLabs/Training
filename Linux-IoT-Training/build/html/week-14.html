What is Linux HA¶

A way for the cluster to communicate about its state and configuration, this is called a cluster messaging layer. A way to automatically manage the resources of the cluster, the cluster resource manager (CRM). Some kind of middle ware between the resource manager and the actual resources, sometimes called resource agents.

http://frankgroeneveld.nl/2014/03/23/basic-concepts-of-high-availability-linux/ https://www.howtoforge.com/high_availability_heartbeat_centos

visible_hostname LITproxy

acl client src 192.168.10.19

acl ban_domains dstdomain “/etc/squid3/ban_domains”

http_access deny ban_domains client

http_access allow client
